API Detail — Empty Tab Treatment

When an API's MCP Config / Schema / Code Snippet / Notes tab has no content, what should the CTA do? Showing the MCP Config tab as the example.

Option A — "Edit to fill in"
⚙️
No MCP config yet
Edit this API entry to add an MCP config manually, or use AI Generate to create one automatically.
Edit API
Option B — Direct "Generate" shortcut
⚙️
No MCP config yet
Generate an MCP config automatically from this API's details using AI.
Edit API
✨ Generate

Option B only shows for editors/admins who can use AI. Suggesters and viewers always see Option A style (no generate shortcut). For Schema, Snippet, and Notes tabs there's no AI shortcut — those always use the "Edit API" path.
