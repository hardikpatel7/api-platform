Implementation notes:
• A shared EmptyState component takes icon, title, description, and an optional actions array — no per-screen custom markup.
• The MCP Config "Generate" button fires the existing generateApiDocsAction server action and refreshes the tab without entering edit mode.
• "Suggest edit" CTA on API detail tabs sets editing=true on the parent page (same as clicking the existing Suggest Edit button in the header).
• Filter empty states ("No APIs match the current filters") keep existing plain-text treatment — they're not onboarding states.
